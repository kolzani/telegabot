# Telegram Username Bot

Бот для подбора свободных пятибуквенных юзернеймов в Telegram.

## Функционал
- Генерирует 5‑символьные юзернеймы (только буквы).
- Проверяет доступность через Telegram API.
- Выводит 5 свободных вариантов по команде `/generate`.
